익명함수
원래함수 형식
function 함수이름 (매개변수) { 실행문; }
익명함수 형식
function (매개변수) { 실행문; } // 이름이 없다.
Immediately Invoked Function Expressions (IIFE) 함수 형식
(function 함수이름 () { 실행문; }) (); // 이름은 없어도 된다. 함수의 정의와 동시에 실행.
